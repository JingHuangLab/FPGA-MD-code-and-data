#

 - build: Compilation files for the FPGA project. The compilation files comply 
    with the standards of the corresponding Vivado platform.

 - script: Scripts related to the project, used for project recompilation or
   automation support scripts. Need Vivado 2022.2 platform compilation tools.

 - constr: Constraint files required for FPGA compilation, including pin 
    constraint files, clock constraint files, etc.

 - bitstream: FPGA programming files, including binary files and output .ltx waveform observation files.
